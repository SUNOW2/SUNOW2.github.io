Node.js中Buffer.copy()用法:

方法说明:

定义:进行不同buffer之间的复制操作，从源buffer复制数据并替换到目标buffer的指定位置

语法:

buffer.copy(targetBuffer, [targetStart], [sourceStart], [sourceEnd])

参数解释:

targetBuffer: 目标buffer,执行复制替换的buffer
targetStart: 目标buffer数据替换的起始位置
sourceStart: 源buffer数据复制的的起始位置
sourceEnd: 源buffer数据复制的结束位置

实例:

var buffer = new Buffer(length);
var pos = 0;
for (var i = 0; i < list.length; i++) {
    var buf = list[i];
    // 将buf中4到6之间的内容复制到buffer(从pos位开始)中
    buf.copy(buffer, pos, 4, 6);
    pos += buf.length;
}

javascript call()方法

By 徐旭

Published 2017-03-19

javascript call()方法

定义

调用一个对象的一个方法，以另一个对象替换当前对象

用法

call([thisObj[,arg1[arg2[,[,…argN]]]]]);

参数

thisObj: 可选项，将被当作当前对象的对象
arg1,arg2,…,argN: 可选项，被传递方法的参数序列

说明

call()方法可以用来代替另一个对象来调用一个方法。call()方法可将一个函数的对象上下文从初始的
上下文转变由thisObj指定的新对象

注:如果没有提供thisObj参数，那么Global对象被用作thisObj

实例一:

var Add = function(a,b){
    console.info(a+b);
}

var Sub = function(a,b){
    console.info(a-b);
}
//输出6，用Add对象替换Sub对象，使用Add上下文来执行方法
Add.call(Sub,3,3);

实例二:

function Sister(){
    this.age = 18;
    this.sayAge = function(){
        document.write("Age:"+this.age);
    }
    this.yaya = function(){
        document.write("yaya...");
    }
}
function Brother(){
    this.age = 25;
    this.sayAge = function(){
        document.write("Age:"+this.age);
    }
    this.hoho = function(){
        document.write("hoho...");
    }
}
var sis = new Sister();
var bro = new Brother();
sis.sayAge.call(bro);

解析:sis.sayAge.call(bro)调用的方法是Sister中的方法，但是其中的this(上下文)却变成了Brother,
用文档的解释就是:只有这个sayAge是sayAge是sis对象的，其执行环境却是bro的

简而言之:sis.sayAge.call(bro)意思就是,单单就sis的sayAge移动到bro对象中执行
