javascript call()方法

定义

调用一个对象的一个方法，以另一个对象替换当前对象

用法

call([thisObj[,arg1[arg2[,[,…argN]]]]]);

参数

thisObj: 可选项，将被当作当前对象的对象
arg1,arg2,…,argN: 可选项，被传递方法的参数序列

说明

call()方法可以用来代替另一个对象来调用一个方法。call()方法可将一个函数的
对象上下文从初始的上下文转变由thisObj指定的新对象

注:如果没有提供thisObj参数，那么Global对象被用作thisObj

实例一:

var Add = function(a,b){
    console.info(a+b);
}

var Sub = function(a,b){
    console.info(a-b);
}
//输出6，用Add对象替换Sub对象，使用Add上下文来执行方法
Add.call(Sub,3,3);

实例二:

function Sister(){
    this.age = 18;
    this.sayAge = function(){
        document.write("Age:"+this.age);
    }
    this.yaya = function(){
        document.write("yaya...");
    }
}
function Brother(){
    this.age = 25;
    this.sayAge = function(){
        document.write("Age:"+this.age);
    }
    this.hoho = function(){
        document.write("hoho...");
    }
}
var sis = new Sister();
var bro = new Brother();
sis.sayAge.call(bro);

解析:sis.sayAge.call(bro)调用的方法是Sister中的方法，但是
其中的this(上下文)却变成了Brother,用文档的解释就是:只有这个
sayAge是sayAge是sis对象的，其执行环境却是bro的

简而言之:sis.sayAge.call(bro)意思就是,单单激昂sis的sayAge
移动到bro对象中执行
